# OpenCV DRS Project

A Computer Vision project developed using Python and OpenCV for cricket ball detection, tracking, and trajectory analysis.

## Features

- Video frame processing
- Cricket ball detection
- Ball tracking across frames
- Image preprocessing
- Trajectory visualization
- Modular project structure

## Technologies

- Python
- OpenCV
- NumPy

## Installation

\\\ash
git clone https://github.com/udbhav968-creator/opencv.git

pip install -r requirements.txt
